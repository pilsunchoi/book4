자료: 2018 FIFA 월드컵 결승전 두 개의 JSON 파일(events, lineups)

1. 개요 (Overview)#

본 문서는 StatsBomb Open Data 형식으로 제공되는 2018 FIFA 월드컵 결승전(프랑스 vs 크로아티아) 경기 데이터의 구조와 필드를 설명합니다. 데이터는 두 개의 JSON 파일로 구성되어 있습니다.

2. 이벤트 파일 (Event Data)#

이벤트 파일은 경기 중 발생한 모든 액션을 시간순으로 기록한 JSON 배열입니다. 각 이벤트는 공통 필드와 이벤트 유형별 전용 필드로 구성됩니다.

2-4. 플레이 패턴 (Play Pattern)#

각 이벤트에는 해당 플레이의 시작 유형이 기록됩니다:

Regular Play · From Corner · From Counter · From Free Kick · From Goal Kick · From Keeper · From Kick Off · From Throw In · Other

2-5. 좌표계 (Coordinate System)#

StatsBomb는 120 × 80 yards 기준 좌표계를 사용합니다. 공격 방향은 항상 왼쪽에서 오른쪽(x: 0→120)이며, y축은 0(top)에서 80(bottom)입니다. 슈팅의 end_location은 3차원(x, y, z)으로 z는 높이를 나타냅니다.

3. 라인업 파일 (Lineup Data)#

라인업 파일은 양팀의 전체 엔트리 정보를 담고 있습니다. 프랑스 23명, 크로아티아 22명으로 총 45명의 선수 데이터가 포함되어 있습니다.

3-1. 필드 구조#

필드명

설명

player_id / player_name

경고/퇴장 카드 목록

포지션 변경 이력 (시간, 사유 포함)

3-2. 포지션 변경 이력 (positions)#

positions 배열은 경기 중 선수의 포지션 변경을 시간순으로 기록합니다. 각 항목에는 position(포지션명), from/to(시간), from_period/to_period(반), start_reason/end_reason(변경 사유: Starting XI, Substitution, Tactical Shift, Final Whistle 등)이 포함됩니다.

예시: Kylian Mbappé는 Right Midfield로 선발 출전한 후, 81분에 Tactical Shift로 Right Center Forward로 변경되어 경기 종료까지 해당 포지션을 유지했습니다.

4. 활용 방안 (Potential Use Cases)#

  • 경기 흐름 분석: timestamp와 possession 시퀀스를 활용하여 경기 흐름을 시각화

  • 패스 네트워크: Pass 이벤트의 passer→recipient 관계로 패스 네트워크 구성

  • xG 모델 검증: Shot 이벤트의 statsbomb_xg와 freeze_frame을 활용한 기대골 분석

  • 수비 분석: Pressure, Duel, Interception, Ball Recovery 데이터로 수비 강도 측정

  • 히트맵 / 시각화: location 좌표를 활용한 슛, 패스, 압박 등의 공간 분석

  • 선수 평가: Lineup의 positions 변경 이력 + 이벤트 데이터를 결합한 개인 퍼포먼스 분석

5. 참고 사항 (Notes)#

  • 데이터 출처: StatsBomb Open Data (statsbomb/open-data)

  • Ball Receipt*의 *는 해당 이벤트가 불완전한 수신(miscontrol)일 수 있음을 나타냅니다.

  • under_pressure (Boolean)는 상대 압박 하에 수행된 이벤트에 추가로 표시됩니다.

  • related_events를 통해 이벤트 간 연결 관계를 파악할 수 있습니다 (예: 패스 ↔ 수신, 슈팅 ↔ 세이브).

  • xG(기대골): 프랑스 1.10, 크로아티아 1.48 — 크로아티아가 xG에서 앞섰으나 실제 결과는 역전.

2. 경기 흐름 분석#

Claude 분석#

경기 흐름 시각화 대시보드를 만들었습니다. 구성 요소는 다음과 같습니다:

🔗 2018 FIFA 월드컵 결승전 경기흐름 대시보드 (https://claude.ai/public/artifacts/e804fc63-135d-41c8-95bb-37013f47b65e)

득점 타임라인 — 6골 모두를 시간축 위에 배치하여 득점 흐름을 한눈에 파악할 수 있습니다.

점유 흐름 스트립 — 182개 점유 시퀀스를 전·후반별로 시각화했습니다. 각 블록의 너비는 이벤트 수에 비례하며, 마우스를 올리면 해당 점유의 상세 정보(시간, 패스 수, 슈팅, xG, 플레이 패턴)를 확인할 수 있습니다. 골은 금색, 슈팅은 흰색 점으로 표시됩니다.

모멘텀 차트 — 5분 이동 평균 기반으로 양팀의 경기 지배력 변화를 보여줍니다. 전반 30분 이후~후반 초반까지 크로아티아가 압도적으로 경기를 지배했으나, 프랑스는 58분-65분 사이 짧은 우세 구간에서 2골을 넣은 것이 잘 드러납니다.

3. 선수 평가#

Claude 분석#

출전한 27명 전원의 개인 퍼포먼스 분석 대시보드를 만들었습니다. 각 선수 카드를 클릭하면 상세 분석이 펼쳐집니다.

🔗 2018 FIFA 월드컵 결승전 선수평가 대시보드 (https://claude.ai/public/artifacts/b878065b-1feb-4837-8014-bd686662baec)

레이더 차트 — 공격, 패스, 운반, 압박, 수비, 볼륨 6개 축으로 Per 90분 기준 선발 선수 대비 백분위를 시각화합니다. Pogba는 공격+수비 양면에서, Brozović는 패스+볼륨에서 극단적인 수치를 보여줍니다.

포지션 변경 이력 — Lineup 데이터의 positions 배열을 타임라인으로 표시합니다. 예를 들어 Mbappé는 81분에 Right Midfield → Right Center Forward로, Griezmann도 같은 시점에 포지션이 변경된 것이 보여 디디에 데샹의 후반 전술 교체를 추적할 수 있습니다.

주요 발견:

  • Brozović — 271터치, 99패스(87성공)로 경기 최다. 크로아티아 빌드업의 절대적 허브

  • Mbappé — 6회 전진 드리블, 245yd 운반 거리로 프랑스 역습의 핵심. xG 0.06에서 1골(결정력 초과 달성)

  • Griezmann — 31회 압박으로 프랑스 최다. 공격수임에도 전방 수비 기여가 압도적

  • Kanté — 54분 출전에 19회 압박, 옐로카드 1장. 교체 이유가 체력+경고 관리였음을 시사
